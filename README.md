# Flickr
exploring the concept of the male gaze through interactive art and eye tracking technology.
an innovative interactive art project that examines gender perception and the male gaze using modern web technologies.
The experience challenges viewers to understand how women are perceived differently through multiple perspectives combining eye tracking technology, interactive visualizations, and feminist theory 

## features
- **Eye-Tracking** - Utilizes eye-tracking technology to capture and analyze viewer gaze patterns
- **Feminist** - Explores feminist perspectives on gender and perception
- **Interactive Art** - Creates immersive, interactive experiences for the audience
- **Male Gaze** - Examines the concept of the male gaze in visual culture

## Technologies Used
- Js
- HTML/CSS
- p5.js - Creative coding and visualization library
- WebGazer.js- Eye-tracking functionality
- Bootstrap
